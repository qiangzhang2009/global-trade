# TradeRoot · 全球贸易导航

> A tree-navigable global B2B channel directory helping Chinese companies find overseas distribution, agency, and procurement channels — organized by continent → region → country → platform.

## English

### What is TradeRoot?

TradeRoot is a professional-grade web directory that helps Chinese manufacturers and export companies discover and connect with overseas distribution channels, agents, and procurement opportunities worldwide.

### Core Features

- **Tree Navigation** — Multi-level drill-down from continent → region → country → channel type
- **900+ Channels** — Official government portals, B2B platforms, distribution networks, and cross-border trade platforms
- **Smart Search** — Instant search across all countries, platforms, and keywords
- **Favorites** — Bookmark channels to your local storage for quick access
- **Bilingual** — Full Chinese / English language toggle
- **Dark & Light Mode** — Professional dark theme and warm light theme
- **Responsive** — Works on desktop and mobile

### How to Use

1. Open `index.html` in any modern browser — no build step required
2. Select a continent from the sidebar to begin
3. Expand regions and countries to explore channels
4. Click any channel card to view details or visit the official site

## 简体中文

### TradeRoot 是什么？

TradeRoot（全球贸易导航）是一款专业级网页目录，帮助中国企业发现并连接全球海外分销渠道、代理商和采购机会。

### 核心功能

- **树形导航** — 从大洲 → 区域 → 国家 → 渠道类型多级展开
- **900+ 渠道** — 涵盖官方政府采购平台、B2B平台、代理分销网络和跨境贸易平台
- **智能搜索** — 跨国家、平台、关键词的即时搜索
- **收藏功能** — 将感兴趣的渠道收藏到本地，随时查看
- **双语切换** — 中文 / English 完整互译
- **深色 / 浅色主题** — 专业深色模式 + 暖色浅色模式
- **响应式布局** — 支持桌面端和移动端

### 使用方式

1. 直接在浏览器中打开 `index.html` — 无需任何构建步骤
2. 从左侧栏选择一个洲开始探索
3. 展开区域和国家，深入查看具体渠道
4. 点击任意渠道卡片查看详情或访问官方网站

### 技术栈

- 纯原生 HTML/CSS/JS — 无框架依赖
- Google Fonts：Instrument Serif、DM Sans、JetBrains Mono
- localStorage 存储收藏和偏好设置
- CSS 自定义属性实现主题切换

| 大洲 | 国家/地区数 | 主要渠道类型 |
